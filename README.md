# PortfolioStudentaBJ
Opis projektu
Aplikacja mobilna portfolio przedstawiająca profil Bartosza Jojko, studenta informatyki zainteresowanego tworzeniem aplikacji webowych i mobilnych, backendem oraz bazami danych. Projekt jest prototypem aplikacji na iOS, docelowo rozwijanej w języku Swift, służącej do prezentacji informacji o autorze, jego umiejętnościach i wybranych projektach w formie nowoczesnego interfejsu inspirowanego systemem iOS.

#Technologie
Docelowa platforma: iOS

Język: Swift (planowany), SwiftUI / UIKit (do wyboru w dalszym etapie)

#Funkcjonalności
Ekran logowania:

Prezentacja ekranu startowego z polem na login/hasło (prototypowo, bez realnej autoryzacji).

Przejście do głównego ekranu profilu po kliknięciu przycisku „Zaloguj się”.
​

Główna strona profilu:

Wyświetlenie imienia i nazwiska (Bartosz Jojko) oraz roli „Student Informatyki – Backend / Mobile”.

Krótki opis zainteresowań: aplikacje webowe i mobilne, bazy danych, architektura systemów, prosty i czytelny UI.

Lista podstawowych umiejętności (Backend: Java, Kotlin, Spring Boot; Frontend: HTML, CSS, React; Narzędzia: Git, GitHub, Docker, Figma).

Zakładka „O mnie”:

Rozszerzony opis autora, jego ścieżki edukacyjnej i zainteresowań związanych z programowaniem oraz tworzeniem aplikacji.

Cel aplikacji: pokazanie się potencjalnym rekruterom/pracodawcom jako junior developer.

Zakładka „Projekty”:

Lista wybranych projektów (np. aplikacje webowe/mobile, projekty uczelniane), każdy z krótkim opisem i technologiami użytymi w projekcie.

Możliwość rozbudowy o linki do repozytoriów GitHub w kolejnych iteracjach.

Zakładka „Kontakt”:

Prezentacja kanałów kontaktu: e‑mail, GitHub, LinkedIn, Instagram, Discord, Telegram, bez możliwości wysyłania wiadomości z poziomu aplikacji.

Krótki nagłówek zachęcający do kontaktu oraz opis celu kontaktu (praktyki, praca juniora, współpraca przy projektach)
